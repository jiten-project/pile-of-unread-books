## アプリの説明

### 説明文（日本語・4000文字以内）

```
積読本管理は、買ったまま読んでいない本（積読）を管理するためのシンプルで使いやすいアプリです。

■ こんな方におすすめ
・本を買っても読まずに積んでしまう方
・どんな本を持っているか把握したい方
・読書の記録をつけたい方
・本の購入費用を把握したい方

■ 主な機能

【バーコードスキャン】
本のバーコード（ISBN）をカメラでスキャンするだけで、書籍情報を自動取得。タイトル、著者、出版社、表紙画像などが自動で入力されます。

【本棚管理】
登録した本を「未読」「読書中」「中断」「読了」のステータスで管理。優先度やタグを設定して、次に読む本を簡単に見つけられます。

【統計・分析】
・積読本の総数と総額
・月別の購入冊数と金額
・読書ステータスの割合
・購入場所や著者の傾向
など、あなたの読書傾向を可視化します。

【リマインダー通知】
定期的に積読本の存在をお知らせ。買った本を忘れずに読み始めるきっかけを作ります。

【データのバックアップ】
本のデータをJSON形式でエクスポート・インポート可能。機種変更時のデータ移行も安心です。

【ダークモード対応】
システム設定に連動したダークモードに対応。目に優しい表示で夜間の利用も快適です。

■ プライバシーへの配慮
・すべてのデータは端末内に保存され、外部サーバーには送信されません
・カメラはバーコードスキャン機能のみに使用されます
・アカウント登録は不要です

シンプルな操作で、あなたの積読本を見える化。
読書ライフをもっと楽しくしましょう！
```

## App Store Connect 設定

### アプリのプライバシー

#### データ収集
**収集するデータはありません**

このアプリは以下の理由により、ユーザーデータを収集しません：
- すべてのデータは端末内のローカルデータベース（SQLite）に保存
- 外部サーバーへのデータ送信なし
- 分析ツールやトラッキングSDKの使用なし
- アカウント登録機能なし

### 使用するAPIとその理由

| API/機能 | 使用理由 |
|----------|----------|
| カメラ | バーコード（ISBN）スキャンのため |
| 通知 | 読書リマインダー通知のため |
| ネットワーク | 書籍情報のAPI検索（OpenBD、Google Books）のため |

### App Review向けの注記

```
このアプリは、書籍のバーコード（ISBN）をスキャンして本の情報を取得し、
ユーザーの「積読」（買ったまま読んでいない本）を管理するアプリです。

【テスト方法】
1. アプリを起動すると、ホーム画面が表示されます
2. 「+」ボタンまたは「本を追加」から本を登録できます
3. 「バーコードスキャン」を選択し、任意の本のISBNバーコードをスキャンしてください
4. 書籍情報が自動で取得されます（日本の書籍はOpenBD API、海外の書籍はGoogle Books APIを使用）
5. 本棚タブで登録した本の一覧が確認できます
6. 統計タブで購入傾向のグラフが表示されます

【カメラ使用について】
カメラはバーコードスキャン機能のみに使用されます。
写真の撮影・保存は行いません。

【通知について】
設定画面で読書リマインダー通知をONにすると、
指定した時間に積読本の存在をお知らせします。
通知は完全にオプションです。

【データについて】
すべてのデータは端末内に保存され、外部サーバーには送信されません。
```
