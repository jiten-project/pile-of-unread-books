## アプリの説明

### 説明文（日本語・4000文字以内）

```
積読は、読書好きの証。
あなたの「いつか読みたい」を大切に育てるアプリです。

積読生活は、買った本を記録して、あなただけの「積読コレクション」を楽しむためのアプリです。罪悪感なんて必要ありません。本との出会いを記録し、いつか読む日を楽しみにしましょう。

■ こんな方におすすめ
・気になる本はつい買ってしまう方
・自分の本棚を眺めるのが好きな方
・「積読」という言葉に親近感を覚える方
・本との出会いを大切にしたい方
・複数デバイスで本の情報を共有したい方

■ 主な機能

【バーコードスキャン】
本のバーコード（ISBN）をカメラでスキャンするだけで、書籍情報を自動取得。タイトル、著者、出版社、表紙画像などが自動で入力されます。本屋さんで買った瞬間に登録できます。

【本棚管理】
登録した本を「未読」「読書中」「中断」「読了」のステータスで管理。「購入動機」も記録できるので、なぜこの本を手に取ったのか、未来の自分へのメッセージを残せます。

【クラウド同期】NEW
Apple IDでサインインすると、複数デバイス間でデータを自動同期。機種変更時も安心です。
・クラウド同期は現在100冊まで対応しています
・ローカル（端末内）保存は無制限です
・オフラインでも通常通り利用可能

【統計・分析】
・積読本の総数と総額
・月別の購入冊数と金額
・読書ステータスの割合
・購入場所や著者の傾向
あなたの読書傾向をグラフで可視化。積み上げた本の数は、あなたの好奇心の証です。

【リマインダー通知】
定期的に積読本の存在をお知らせ。「そういえばこの本があった！」という再発見の楽しみをお届けします。

【データのバックアップ】
本のデータをJSON形式でエクスポート・インポート可能。大切なコレクションを安全に保管できます。

【ダークモード対応】
システム設定に連動したダークモードに対応。夜の読書タイムにも最適です。

■ プライバシーへの配慮
・クラウド同期を使わない場合、すべてのデータは端末内のみに保存されます
・クラウド同期を使う場合も、データはあなたのApple IDに紐づけられ、本人のみがアクセスできます
・カメラはバーコードスキャン機能のみに使用されます
・サインインせずに利用することもできます

積読は、未来の自分への贈り物。
あなたの積読ライフを、もっと楽しく。
```

## App Store Connect 設定

### アプリのプライバシー

#### データ収集（クラウド同期機能使用時）

**クラウド同期を使用しない場合**：収集するデータはありません

**クラウド同期を使用する場合**：以下のデータを収集・保存します

| データの種類 | 収集目的 | 第三者との共有 |
|-------------|---------|---------------|
| Apple ID識別子 | ユーザー認証、データの紐づけ | なし |
| メールアドレス | ユーザー識別（非公開設定可能） | なし |
| 書籍データ | クラウド同期機能の提供 | なし |

- データはSupabase（クラウドデータベース）に保存
- Row Level Security（RLS）により本人のみがアクセス可能
- 分析ツールやトラッキングSDKは使用していません
- クラウド同期は任意の機能です

### 使用するAPIとその理由

| API/機能 | 使用理由 |
|----------|----------|
| カメラ | バーコード（ISBN）スキャンのため |
| 通知 | 読書リマインダー通知のため |
| ネットワーク | 書籍情報のAPI検索（OpenBD、Google Books）、クラウド同期のため |
| Sign in with Apple | クラウド同期機能のユーザー認証のため |

### App Review向けの注記

```
このアプリは、書籍のバーコード（ISBN）をスキャンして本の情報を取得し、
ユーザーの「積読」（買ったまま読んでいない本）を管理するアプリです。

【テスト方法】
1. アプリを起動すると、ホーム画面が表示されます
2. 「+」ボタンまたは「本を追加」から本を登録できます
3. 「バーコードスキャン」を選択し、任意の本のISBNバーコードをスキャンしてください
4. 書籍情報が自動で取得されます（日本の書籍はOpenBD API、海外の書籍はGoogle Books APIを使用）
5. 本棚タブで登録した本の一覧が確認できます
6. 統計タブで購入傾向のグラフが表示されます

【クラウド同期機能について（v1.1.0新機能）】
・設定画面から「Apple IDでサインイン」を選択するとクラウド同期が有効になります
・Sign in with Appleを使用してユーザー認証を行います
・データはSupabase（クラウドデータベース）に保存されます
・クラウドに同期できる書籍は現在100冊までです
・ローカル（端末内）保存は無制限です
・サインインせずに、ローカルのみで使用することも可能です
・オフラインでもアプリは通常通り動作し、オンライン復帰時に自動同期されます

【カメラ使用について】
カメラはバーコードスキャン機能のみに使用されます。
写真の撮影・保存は行いません。

【通知について】
設定画面で読書リマインダー通知をONにすると、
指定した時間に積読本の存在をお知らせします。
通知は完全にオプションです。

【データについて】
・クラウド同期を使わない場合：すべてのデータは端末内のみに保存されます
・クラウド同期を使う場合：データはApple IDに紐づけられ、Supabaseに保存されます
・ユーザーは自分のデータにのみアクセスできます（Row Level Security適用）
```

## 暗号化に関する情報

### Export Compliance（輸出コンプライアンス）

**ITSAppUsesNonExemptEncryption: false**

このアプリは以下の理由により、輸出規制対象の暗号化を使用しません：
- HTTPSはオペレーティングシステム標準のAPI呼び出しのみ使用（Supabase、OpenBD、Google Books API）
- Sign in with AppleはApple標準の認証フレームワークを使用
- カスタム暗号化アルゴリズムは使用していません
- ローカルデータの暗号化は行っていません
